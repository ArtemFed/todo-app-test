# todo-app-test
### Задание:
Сделать вэб приложеие для работы с пользователями и их списками задач.
#### Функционал:
1) Регистрация пользователя
2) Авторизация пользователя
3) Создание списков задач
4) Создание задач
#### Технологии
*  Golang (gin, JWT)
*  Postgres
*  Паттерн: Чистая архитектура по Роберту Мартину
